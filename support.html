Common Questions

My sets aren't parsing correctly

Set understands natural shorthand like "BS 5x3 @ 100kg" or "deadlift 1 rep 180 kg". Try using common abbreviations (BS, DL, OHP, BP) followed by sets, reps, and weight. If a parse looks wrong, you can edit it in the preview before saving.

iCloud sync isn't working

Make sure iCloud is enabled for Set in your device settings: Settings > [Your Name] > iCloud > Apps Using iCloud. Both devices must be signed into the same Apple ID. Sync may take a few moments over a cellular connection.

How do I delete a movement?

Go to the Lifts tab, tap the movement you want to remove, then use the delete option at the bottom of the movement detail screen. This will remove the movement and all associated sets.

How do I reset all my data?

Open the Profile tab, scroll to the bottom, and tap "Reset All Data". This permanently deletes all local data including lifts, sets, and settings. This action cannot be undone.

Can I export my data?

Yes. Set supports Markdown export of your max lifts via the share sheet. Open the Profile tab and use the export option to share your data as formatted text.

What units does Set support?

Set supports both kilograms (kg) and pounds (lb). You can specify the unit when logging, and Set will parse it correctly. The default unit can be set in your preferences.
